## 常微分方程为： x'(t)=-2x(t)+t
<!--more-->
```
clear;
% 将方程转换为: ax'=bx + ct
a=1.;
b=-2.;
c=1.;
% 初始时间
tinit= 0.;
tmax=5.;
% 步数
maxt = 3000;
dt = (tmax-tinit)/maxt;
% 初始条件
x(2)=1.;
x(1)=1.0-dt*((b*x(2)+c*(2)*dt)/a);
t(2)=tinit;
% 迭代求解
for j=2:(maxt+1);
x(j+1)=x(j-1)+2.*dt*((b*x(j)+c*(j)*dt)/a);
t(j+1)=tinit+(j-1)*dt;
end;
plot(t,x)
title('Midpoint Method')
xlabel('T')
ylabel('X(t)')
```
